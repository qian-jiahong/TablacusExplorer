추가 기능이 지원되는 탭 형식의 파일 관리자입니다.
Tablacus Explorer

저자: Gaku


Tablacus Explorer의 기능:

* 탭 인터페이스
* 추가 지원
* 완전히 휴대 가능하며 설치 필요 없음
* 다중 언어 및 유니코드 지원
* 사용자 정의 가능한 연결, 메뉴, 키, 마우스 제스처, 별칭
* 오픈소스
* 64비트 (TE64.exe/x64 Edition) 및 32비트 (TE32.exe/x86 Edition) 버전 사용 가능


설치:

* ZIP 파일을 임의의 폴더로 압축 풀기
* Tablacus Explorer (te64.exe/te32.exe)

- Program Files 디렉터리에 Tablacus Explorer를 설치한 경우:

* 설정은 "%AppData%\tablacus\explorer\config"에 저장됩니다.
* 옵션 열기
* "구성 폴더 초기화"를 누릅니다


제거:

* tablacus 폴더를 삭제합니다.

- Program Files 디렉터리에 Tablacus Explorer를 설치한 경우:

* "%AppData%\tablacus" 폴더도 삭제합니다.
